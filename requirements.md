# Requirements
## 本ドキュメントの概要
アプリの要件を記す

##　要件
- 画面中央にADD ITEMボタンとリストがある
- ADD ITEMボタンを押すと入力フォーム画面に遷移する
- 入力フォームはタイトル、カテゴリ、画像の３つ入力要素をもつ
- タイトルは必須、他は任意
- タイトルはテキスト入力
- カテゴリは選択肢入力、選択肢はビール、コーヒーの２つ
- 画像はデバイスからアップロードで添付
- 入力フォームは右下の方に保存ボタンがある
- 保存ボタンを押すと入力内容が保存される
- 入力した内容は画面のリストに追加される
